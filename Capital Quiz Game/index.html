About This Game

Capital Quiz is the result of a coding challenge presented in Jonas Schmedtmann's Udemy class on Javascript.

While the challege asked students to create a mini-quiz game that played in a browser development console, I wanted to apply everything we'd learned so far to make a browser-based game.
